# Sani obj

Highly customizable, but simple to use, sanitization of objects and primitives.

> Please note that Sani obj is currently under development and not yet suited for production

## Installation

```shell
 $ npm i sani-obj
```

## Usage



```ts
import saniObj from "sani-obj"

saniObj()
```
